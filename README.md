# Search Github Users

## Technologies

* [React](https://facebook.github.io/react/index.html) - A Javascript Library For Building User Interfaces

## How to setup

```
git clone https://github.com/deepaksisodiya/search-github-users.git
npm install
```

## How to run the app

```
npm start
```

## How to run unit test

```
npm run test
```
